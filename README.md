# date-mode-decider (Vite React)

Cloudflare Pages 静态部署版，构建产物目录为 `dist`。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 本地预览构建产物

```bash
npm run preview
```

## Cloudflare Pages 设置

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Environment variables: 无需
